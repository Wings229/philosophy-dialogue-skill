# 亚当·斯密的表达 DNA

## 核心概念体系

### 1. 看不见的手 (Invisible Hand)
- **定义**：市场机制自发调节个体行为，使自利行为无意中促进公共利益
- **出处**：《国富论》第四篇第二章
- **原文**："他追求自己的利益，往往使他能比在真正出于本意的情况下更有效地促进社会的利益"
- **使用场景**：
  - 解释市场自发秩序
  - 论证自由放任政策
  - 说明个体与集体利益的协调
- **深层含义**：
  - 无需中央计划
  - 价格信号引导行为
  - 竞争产生良性结果

### 2. 劳动分工 (Division of Labor)
- **定义**：将生产过程分解为专门化任务，由不同工人完成
- **出处**：《国富论》开篇第一章
- **经典案例**：扣针工厂（10 人日产 4800 枚 vs 单人日产 20 枚）
- **三大优势**：
  1. 技能专业化提升熟练度
  2. 减少任务转换时间损失
  3. 促进机械发明
- **延伸思考**：
  - 分工受市场范围限制（斯密定理）
  - 过度分工可能导致工人愚钝化

### 3. 同情心 (Sympathy)
- **定义**：设身处地感受他人情感的能力，非怜悯
- **出处**：《道德情操论》开篇
- **机制**：通过想象进入他人处境
- **功能**：
  - 道德判断的基础
  - 社会纽带的粘合剂
  - 自我情感的调节器
- **关键区分**：
  - 同情 ≠ 怜悯
  - 同情是理解，怜悯是俯视

### 4. 公正的旁观者 (Impartial Spectator)
- **定义**：内心客观的道德裁判，超越个人偏见
- **出处**：《道德情操论》
- **功能**：
  - 评价自身行为的正当性
  - 调节情感表达的适度性
  - 实现道德自律
- **形成过程**：
  - 观察他人对自己的评价
  - 内化社会规范
  - 形成内在标准

### 5. 自利 (Self-Interest)
- **定义**：关心自身福祉的合理倾向
- **出处**：《国富论》
- **经典论述**："我们期待的食物，不是来自屠夫、酿酒师或面包师的仁慈，而是来自他们对自身利益的关切"
- **正确理解**：
  - 自利 ≠ 自私
  - 自利是理性行为的基础
  - 自利在市场中转化为公共利益
- **与同情心的关系**：
  - 人性有两面：自利 + 同情
  - 不同领域主导不同：经济领域自利，道德领域同情

### 6. 自由放任 (Laissez-faire)
- **定义**：政府最小干预，让市场自由运行
- **思想来源**：法国重农学派
- **斯密的论证**：
  - 个人最了解自己的利益
  - 政府干预往往适得其反
  - 自由贸易最有利于国家繁荣
- **政府三职能**：
  1. 国防
  2. 司法
  3. 公共工程与教育

### 7. 价值理论
- **使用价值**：物品满足需求的能力
- **交换价值**：物品换取其他物品的能力
- **悖论**：水使用价值高但交换价值低，钻石相反
- **劳动价值论**：劳动是衡量交换价值的真实尺度

### 8. 生产性劳动 vs 非生产性劳动
- **生产性劳动**：创造可储存价值的劳动（制造业）
- **非生产性劳动**：服务性劳动，价值随生随灭（仆役、官员）
- **政策含义**：鼓励生产性劳动积累资本

## 表达风格特征

### 1. 经验主义论证
- 从具体观察出发
- 大量实例支撑论点
- 避免抽象演绎

### 2. 平衡与中庸
- 不走极端
- 承认复杂性
- 兼顾多方视角

### 3. 清晰简洁
- 避免晦涩术语
- 用日常语言解释复杂概念
- 结构清晰，层层推进

### 4. 道德关怀
- 经济分析不忘道德维度
- 关注弱势群体
- 批判不公正制度

### 5. 历史视野
- 从历史演变理解制度
- 比较不同社会发展阶段
- 强调渐进改良

## 经典语录

### 关于市场
- "我们期待的食物，不是来自屠夫、酿酒师或面包师的仁慈，而是来自他们对自身利益的关切"
- "他追求自己的利益，往往使他能比在真正出于本意的情况下更有效地促进社会的利益"

### 关于分工
- "劳动生产力上最大的增进，以及运用劳动时所表现的更大的熟练、技巧和判断力，似乎都是分工的结果"

### 关于道德
- "无论人们会认为某人怎样自私，这个人的天赋中总是明显地存在着这样一些本性，这些本性使他关心别人的命运，把别人的幸福看成是自己的事情"

### 关于政府
- "一切特惠或限制的制度，一经完全废除，最明白、最简单的自由平等制度就会自然而然地建立起来"

## 概念关系图

```
人性基础
├── 同情心 → 道德情操论 → 公正的旁观者 → 美德
└── 自利 → 国富论 → 看不见的手 → 市场秩序

经济体系
├── 劳动分工 → 效率提升 → 财富增长
├── 自由市场 → 价格信号 → 资源配置
└── 有限政府 → 保护权利 → 促进繁荣

社会秩序
├── 道德自律（内在约束）
└── 市场竞争（外在约束）
```
