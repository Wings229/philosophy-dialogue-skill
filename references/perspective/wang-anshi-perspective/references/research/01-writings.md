# 王安石著作研究

## 一、《临川先生文集》

### 概述
- 原名：《临川先生文集》，又称《王文公文集》
- 卷数：一百卷
- 内容：诗、文、奏议、书信等
- 版本：南宋绍兴年间刊刻，后世有多个版本

### 文集结构
1. **诗**（约 1500 首）
   - 早期诗作：清新自然，如《泊船瓜洲》
   - 中期诗作：关注民生，如《河北民》
   - 晚期诗作：沉郁顿挫，如《桂枝香·金陵怀古》

2. **文**（散文、议论文）
   - 政论文：《上仁宗皇帝言事书》《本朝百年无事札子》
   - 书信：《答司马谏议书》
   - 记游文：《游褒禅山记》

3. **奏议**
   - 变法相关奏章：《乞制置三司条例》《乞免京东西路青苗钱札子》
   - 用人奏章：《乞馆职罢诗赋札子》

## 二、重要变法奏议

### 1. 《上仁宗皇帝言事书》（1058 年）
- 背景：任三司度支判官时上奏
- 核心观点：
  - "顾内则不能无以社稷为忧，外则不能无惧于夷狄"
  - 提出"改易更革"的必要性
  - 强调人才选拔与培养

### 2. 《本朝百年无事札子》（1068 年）
- 背景：宋神宗即位后召见王安石
- 核心观点：
  - 表面无事，实则危机四伏
  - "赖非夷狄昌炽之时，又无尧、汤水旱之变，故天下无事过于百年"
  - 呼吁变法图强

### 3. 熙宁变法相关奏议
- 《乞制置三司条例司》：设立变法机构
- 《青苗法奏》：推行青苗法
- 《募役法奏》：改革差役制度
- 《保甲法奏》：建立地方武装
- 《市易法奏》：政府调控市场
- 《方田均税法奏》：重新丈量土地

## 三、诗词代表作

### 著名诗作
1. **《泊船瓜洲》**
   ```
   京口瓜洲一水间，钟山只隔数重山。
   春风又绿江南岸，明月何时照我还。
   ```

2. **《元日》**
   ```
   爆竹声中一岁除，春风送暖入屠苏。
   千门万户曈曈日，总把新桃换旧符。
   ```

3. **《登飞来峰》**
   ```
   飞来山上千寻塔，闻说鸡鸣见日升。
   不畏浮云遮望眼，自缘身在最高层。
   ```

4. **《梅花》**
   ```
   墙角数枝梅，凌寒独自开。
   遥知不是雪，为有暗香来。
   ```

5. **《河北民》**
   ```
   河北民，生近二边长苦辛。
   家家养子学耕织，输与官家事夷狄。
   ```

### 词作
1. **《桂枝香·金陵怀古》**
   ```
   登临送目，正故国晚秋，天气初肃。
   千里澄江似练，翠峰如簇。
   归帆去棹残阳里，背西风，酒旗斜矗。
   彩舟云淡，星河鹭起，画图难足。
   
   念往昔，繁华竞逐，叹门外楼头，悲恨相续。
   千古凭高对此，漫嗟荣辱。
   六朝旧事随流水，但寒烟衰草凝绿。
   至今商女，时时犹唱，后庭遗曲。
   ```

2. **《菩萨蛮·数间茅屋闲临水》**
   ```
   数间茅屋闲临水，窄衫短帽垂杨里。
   花是去年红，吹开一夜风。
   
   梢梢新月偃，午醉醒来晚。
   何物最关情，黄鹂三两声。
   ```

## 四、文章风格特点

### 1. 政论文特点
- **逻辑严密**：层层推进，论证充分
- **语言简劲**：不尚华丽，重在说理
- **气势磅礴**：如江河直下，不可阻挡
- **切中时弊**：直面问题，不避权贵

### 2. 散文特点
- **清新自然**：如《游褒禅山记》
- **哲理深刻**：借景抒情，寓理于事
- **结构精巧**：起承转合，章法严谨

### 3. 诗词特点
- **早期**：清新明快，受欧阳修影响
- **中期**：关注现实，反映民生
- **晚期**：沉郁顿挫，意境深远
- **用典精当**：化用前人诗句，自然贴切
- **炼字精准**：如"春风又绿江南岸"之"绿"字

## 五、文学地位

### 唐宋八大家之一
- 与韩愈、柳宗元、欧阳修、苏洵、苏轼、苏辙、曾巩并称
- 散文成就：政论文最为突出
- 诗词成就：宋诗开山祖之一

### 对后世影响
- 开创"荆公体"诗风
- 影响南宋陆游、辛弃疾等
- 政论文为后世改革家所推崇
