## 思想发展脉络

### 早期（前 697 年前）
- 贫困经历 → 理解民间疾苦
- 与鲍叔牙的友谊 → 知人善任的价值观
- 从商经历（传说） → 经济思维的萌芽

### 中期（前 685-667 年）
- 内政改革 → 制度设计能力
- 经济改革 → 轻重术、官山海的实践
- 「尊王攘夷」 → 政治战略

### 晚期（前 667-645 年）
- 霸业稳固 → 制度化的思考
- 临终谏言 → 识人用人的智慧
- 思想成熟 → 《管子》核心篇章形成

## 关键转折点

1. **前 697 年：从囚徒到相国**
   - 鲍叔牙的推荐
   - 齐桓公的胸襟
   - 管仲的治国才能

2. **前 685-680 年：改革启动**
   - 内政：叁其国而伍其鄙（行政改革）
   - 经济：相地而衰征（税收改革）
   - 军事：作内政而寄军令（兵农合一）

3. **前 667 年：霸业初成**
   - 经济改革成效显现
   - 齐国国力大增
   - 诸侯开始归附

4. **前 645 年：临终谏言**
   - 识别小人（易牙、竖刁、开方）
   - 齐桓公未采纳 → 死后齐国大乱
   - 显示管仲的识人智慧

## 思想影响时间线

| 时期 | 影响 |
|------|------|
| 战国 | 商鞅变法受管子影响（法家制度化） |
| 西汉 | 桑弘羊盐铁专卖（官山海的直接继承） |
| 唐代 | 刘晏理财（轻重术的运用） |
| 宋代 | 王安石变法（青苗法、市易法等） |
| 明代 | 张居正改革（一条鞭法等） |
| 现代 | 国家宏观调控、战略储备制度 |

## 研究备注

- 管子的确切生卒年有争议，采用通行说法（前 723-645）
- 《管子》成书复杂，含战国至汉代学者补充内容
- 核心思想（仓廪实知礼节、轻重术、官山海）可确认为管子原创
- 部分篇章（如《弟子职》）可能为后人托名
