# 曹操表达 DNA 研究

## 一、核心名言原文与出处

### 1.1「宁我负人，毋人负我」
**原文**：「宁我负人，毋人负我」
**出处**：《三国志》裴松之注引《魏书》《世语》
**背景**：杀吕伯奢后
**可信度**：★★★☆☆（可能为后世加工）
**分类**：演义成分

**句式特征**：
- 对仗结构：「宁…毋…」
- 价值排序：自我优先
- 四字格，简洁有力

### 1.2「老骥伏枥，志在千里」
**原文**：「老骥伏枥，志在千里。烈士暮年，壮心不已」
**出处**：《龟虽寿》
**背景**：晚年言志
**可信度**：★★★★★
**分类**：史实

**句式特征**：
- 比喻修辞：老骥自比
- 对比强烈：伏枥 vs 千里
- 积极昂扬

### 1.3「周公吐哺，天下归心」
**原文**：「周公吐哺，天下归心」
**出处**：《短歌行》
**背景**：求贤
**可信度**：★★★★★
**分类**：史实

**句式特征**：
- 用典：周公典故
- 因果关系：吐哺→归心
- 政治抱负表达

### 1.4「唯才是举」
**原文**：「唯才是举，吾得而用之」
**出处**：《求贤令》
**背景**：人才政策
**可信度**：★★★★★
**分类**：史实

**句式特征**：
- 四字成语
- 排他性：「唯」
- 行动导向：「举」「用」

### 1.5「设使国家无有孤，不知当几人称帝，几人称王」
**原文**：「设使国家无有孤，不知当几人称帝，几人称王」
**出处**：《让县自明本志令》
**背景**：回应篡位质疑
**可信度**：★★★★★
**分类**：史实

**句式特征**：
- 假设句：「设使」
- 反问语气
- 自信甚至自负

## 二、句式特征分析

### 2.1 偏好四字格
| 例子 | 出处 |
|------|------|
| 老骥伏枥 | 《龟虽寿》 |
| 志在千里 | 《龟虽寿》 |
| 烈士暮年 | 《龟虽寿》 |
| 壮心不已 | 《龟虽寿》 |
| 唯才是举 | 《求贤令》 |
| 天下归心 | 《短歌行》 |
| 对酒当歌 | 《短歌行》 |
| 人生几何 | 《短歌行》 |

**特征**：简洁、节奏感强、易传诵

### 2.2 善用对比
| 对比项 | 例子 |
|--------|------|
| 短暂 vs 永恒 | 「譬如朝露」vs「天下归心」 |
| 老 vs 志 | 「老骥」vs「千里」 |
| 天 vs 人 | 「不但在天」vs「可得永年」 |
| 负人 vs 负我 | 「宁我负人，毋人负我」 |

**特征**：强化张力，突出主题

### 2.3 常用设问与反问
| 类型 | 例子 |
|------|------|
| 设问 | 「何以解忧？唯有杜康」 |
| 反问 | 「何时可掇？」 |
| 反问 | 「何枝可依？」 |
| 反问 | 「不知当几人称帝，几人称王」 |

**特征**：增强语气，引导思考

### 2.4 用典频繁但自然
| 典故 | 出处 | 曹操使用 |
|------|------|----------|
| 周公吐哺 | 《史记》 | 「周公吐哺，天下归心」 |
| 青青子衿 | 《诗经》 | 「青青子衿，悠悠我心」 |
| 呦呦鹿鸣 | 《诗经》 | 「呦呦鹿鸣，食野之苹」 |
| 齐桓公 | 《史记》 | 「齐桓其何以霸世」 |

**特征**：典故服务于现实目的，不炫学

## 五、表达背后的思维模式

### 5.1 现实主义
- 不回避人生短暂（「譬如朝露」）
- 但主张积极作为（「壮心不已」）
- 接受复杂人性（「进取之士，未必能有行」）

### 5.2 实用主义
- 用典为现实服务
- 语言简洁，不尚空谈
- 目标导向明确

### 5.3 自信甚至自负
- 「天下英雄，唯使君与操耳」
- 「不知当几人称帝，几人称王」
- 但这种自信有实力支撑

### 5.4 复杂性与矛盾性
- 既说「宁我负人」，又关心婢妾生计
- 既「挟天子」，又终身不称帝
- 既杀孔融，又厚待关羽
