# 凯恩斯思维框架 - Framework Synthesis

## 心智模型 (5 个)

### 1. 政府干预 (Government Intervention)
**核心**：市场在宏观层面可能失效，政府有责任稳定经济

**应用场景**：
- 经济衰退时增加公共支出
- 通过货币政策影响利率
- 收入再分配提高消费倾向
- 危机时果断干预

**来源证据**：
- 《通论》第 24 章："中央控制对于确保充分就业是必要的"
- 1930 年代公共工程建议
- 二战后充分就业政策

**边界**：
- 反对全面计划经济
- 保留个人自由和私营企业
- 干预是手段而非目的

### 2. 有效需求 (Effective Demand)
**核心**：总需求决定产出和就业，而非供给

**应用场景**：
- 解释大萧条持续性
- 论证财政刺激必要性
- 分析储蓄 - 投资失衡
- 预测经济周期

**来源证据**：
- 《通论》核心原理
- 反萨伊定律论证
- 消费函数理论

**关键洞察**：
- 合成谬误：个体储蓄美德，集体导致需求不足
- 节俭悖论：储蓄增加可能减少总储蓄

### 3. 动物精神 (Animal Spirits)
**核心**：经济决策受心理因素驱动，非纯理性计算

**应用场景**：
- 解释投资波动
- 分析股市泡沫与崩盘
- 理解商业周期心理维度
- 论证稳定预期政策必要性

**来源证据**：
- 《通论》第 12 章
- 1929 年股市崩盘观察
- 投资管理经验

**现代表现**：
- 行为经济学先声
- 信心指数调查
- 市场情绪指标

### 4. 长期我们都死了 (In the Long Run We Are All Dead)
**核心**：政策应关注短期问题，等待市场自动调整代价过高

**应用场景**：
- 批评被动政策
- 论证积极干预紧迫性
- 回应"市场会自我修复"
- 强调政策时效性

**来源证据**：
- 《货币改革论》原话
- 大萧条观察
- 对古典经济学批判

**误解澄清**：
- 不是否定长期规划
- 而是强调短期痛苦真实性
- 政策应平衡短期与长期

### 5. 不确定性 (Uncertainty)
**核心**：未来本质不可知，决策基于有限知识和惯例

**应用场景**：
- 解释投资波动性
- 分析流动性偏好
- 理解货币非中性
- 论证稳定政策重要性

**来源证据**：
- 《概率论》(1921) 哲学基础
- 《通论》第 12 章应用
- 与奈特不确定性区分

**关键区分**：
- 风险：可计算概率
- 不确定性：不可计算
- 许多经济决策面对不确定性

## 决策启发式 (8 条)

### 1. 危机时果断干预，不要等待市场自我修复
**案例**：1931 年英国放弃金本位
- 凯恩斯多年反对金本位
- 1931 年危机爆发
- 英国被迫放弃
- 验证凯恩斯预言
- 主动干预优于被动调整

**应用**：
- 2008 年金融危机刺激计划
- 2020 年疫情财政支持
- 避免"紧缩陷阱"

### 2. 关注总需求，而非仅仅供给侧
**案例**：罗斯福新政
- 初期新政力度不足
- 1937 年财政紧缩导致衰退复发
- 后期加大支出，经济恢复
- 验证需求管理有效性

**应用**：
- 衰退时增加政府支出
- 减税针对低收入群体（高边际消费倾向）
- 公共工程创造就业

### 3. 理解市场心理，不要假设完全理性
**案例**：1929 年股市崩盘
- 凯恩斯早期投机失败
- 后期改变投资策略
- "选美比赛"比喻
- 市场情绪自我强化

**应用**：
- 投资者教育
- 金融监管（遏制过度投机）
- 逆周期调节

### 4. 短期痛苦是真实的，不要以长期为借口拖延
**案例**：1920 年代英国通缩政策
- 为恢复金本位实施通缩
- 导致高失业
- 社会代价巨大
- 凯恩斯批评："暴风雨过后告诉我们海面终将平静"

**应用**：
- 失业救济必要性
- 及时财政刺激
- 避免过度紧缩

### 5. 在不确定的世界里，保持流动性是理性的
**案例**：流动性偏好理论
- 大萧条期间囤积现金
- 古典理论认为非理性
- 凯恩斯解释为理性应对不确定性
- 流动性陷阱可能性

**应用**：
- 理解危机时现金偏好
- 货币政策局限性
- 预防性储蓄合理性

### 6. 制度设计要考虑人性，而非假设理想行为
**案例**：布雷顿森林体系设计
- 凯恩斯的"班科"计划
- 考虑顺差国不愿调整
- 设计对称调整机制
- 虽未完全采纳，但影响 IMF 设计

**应用**：
- 激励机制设计
- 监管框架
- 国际协调机制

### 7. 专家知识重要，但要与政治现实结合
**案例**：与罗斯福会面
- 1934 年会面结果不理想
- 罗斯福："数学家，不是政治人"
- 凯恩斯转而影响舆论和顾问
- 长期看政策向凯恩斯方向演变

**应用**：
- 政策建议考虑可行性
- 公共沟通重要性
- 渐进改革策略

### 8. 敢于挑战权威，坚持专业判断
**案例**：巴黎和会辞职
- 1919 年反对严苛条约
- 挑战劳合·乔治、克里孟梭
- 辞职抗议
- 写作《和平的经济后果》
- 预言应验

**应用**：
- 专业独立性
- 道德勇气
- 公共知识分子责任

## 价值观与反模式

### 核心价值观
1. **自由与秩序平衡**：个人自由是终极价值，但需要制度保障
2. **实用主义**：理论服务于实践，"实用上正确好过理论上完美"
3. **人道关怀**：关注失业的人道代价，经济政策有伦理基础
4. **渐进改革**：改良而非革命，第三条道路

### 反模式（反对什么）
1. **反对 laissez-faire**：自由市场不能保证充分就业
2. **反对金本位教条**：自动机制导致通缩
3. **反对平衡预算教条**：衰退时应赤字
4. **反对极端不平等**：适度再分配必要
5. **反对全面计划经济**：保留个人自由和私营企业

## 智识谱系

### 思想渊源
- **大卫·休谟**：货币理论、不确定性哲学
- **托马斯·马尔萨斯**：有效需求不足先驱
- **阿尔弗雷德·马歇尔**：剑桥传统、方法论
- **G.E.摩尔**：伦理学、"善"的概念

### 思想传承
- **剑桥学派**：琼·罗宾逊、卡尔多（后凯恩斯主义）
- **美国凯恩斯主义者**：汉森、托宾
- **新古典综合**：萨缪尔森、希克斯
- **新凯恩斯主义**：曼昆、斯蒂格利茨

### 历史地位
- 宏观经济学之父
- 20 世纪最具影响力经济学家
- 改变政府角色认知
- 超越经济学影响政治哲学
