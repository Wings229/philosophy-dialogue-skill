# 范蠡资料来源

## 一手史料

### 1. 《史记》
- **作者**：司马迁（西汉）
- **篇目**：
  - 《越王勾践世家》：范蠡辅佐勾践的完整记载
  - 《货殖列传》：范蠡的商业活动和经济思想
- **可靠性**：最高，最权威的一手资料
- **引用方式**：优先引用

### 2. 《国语》
- **篇目**：《越语》
- **内容**：越国历史，范蠡的政治主张
- **可靠性**：高，先秦史料
- **特点**：对话体，保留原始言论

### 3. 《越绝书》
- **作者**：袁康、吴平（东汉）
- **内容**：越国历史，吴越争霸
- **可靠性**：中等，部分内容为后世补充
- **特点**：地方史志，细节丰富

### 4. 《吴越春秋》
- **作者**：赵晔（东汉）
- **内容**：吴越两国历史
- **可靠性**：中等，有文学加工成分
- **特点**：故事性强

> 「范蠡事越王勾践，既苦身戮力，与勾践深谋二十馀年，竟灭吴」
> 「范蠡以为大名之下，难以久居，且勾践为人可与同患，难与处安」
> 「乃装其轻宝珠玉，自与其私徒属乘舟浮海以行，终不反」

> 「范蠡既雪会稽之耻...乃乘扁舟浮于江湖」
> 「十九年之中三致千金」
