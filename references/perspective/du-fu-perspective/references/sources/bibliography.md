# 杜甫研究资料来源

## 一手资料

### 诗集
- **《杜工部集》**：60 卷（原集已佚，后人辑录）
- **《杜工部小集》**：樊晃编（768 年），最早诗集
- **《杜工部集》**：王洙编（北宋），20 卷，后世通行本

### 史书
- **《旧唐书·杜甫传》**
- **《新唐书·杜甫传》**
- **《资治通鉴》**：安史之乱相关记载

### 唐代文献
- **《全唐诗》**：收录杜甫诗 1400 余首
- **《文苑英华》**：收录杜甫部分诗文
- **唐人选唐诗**：部分收录杜甫诗

## 二手资料（权威研究）

### 古代注本
1. **《钱注杜诗》**：钱谦益（清），以史证诗
2. **《杜诗详注》**：仇兆鳌（清），集大成注释本，50 卷
3. **《读杜心解》**：浦起龙（清）
4. **《杜诗镜铨》**：杨伦（清）

### 近现代研究
1. **《杜甫传》**：冯至（1951 年），经典传记
2. **《杜甫研究》**：萧涤非（1950 年代），系统研究
3. **《杜甫诗选》**：冯至、余冠英选注
4. **《杜诗镜铨批抄》**：程千帆
5. **《杜甫》**：闻一多

### 海外研究
1. **《中国传统诗歌与诗学》**：宇文所安（Stephen Owen）
2. **《杜甫：中国最伟大的诗人》**：洪业（William Hung）
3. 日本：吉川幸次郎《杜甫诗注》
4. 韩国：多种杜诗研究

## 研究方法论

1. **以诗证史**：从杜甫诗歌中提取生平信息
2. **史诗互证**：用史书验证诗歌背景
3. **文本细读**：深入分析诗歌语言与意象
4. **比较研究**：与李白、王维等同时代诗人对比
5. **接受史研究**：梳理历代对杜甫的评价变化
