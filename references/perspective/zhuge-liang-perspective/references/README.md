## 核心发现总结

### 诸葛亮思维框架核心

#### 5 个心智模型
1. **隆中对·战略规划**：全局分析，长期路径
2. **鞠躬尽瘁·极致责任**：对使命绝对忠诚
3. **以法治蜀·制度优先**：赏罚分明，法不容情
4. **淡泊明志·内在定力**：不为外物所动
5. **谨慎持重·风险厌恶**：稳扎稳打，不冒大险

#### 8 条决策启发式
1. 未出茅庐先定三分
2. 知人善任但亲自把关
3. 法不容情
4. 联弱抗强
5. 以攻代守
6. 诫子以俭
7. 先人后己
8. 宁拙勿巧

#### 表达 DNA
- 对偶句式
- 因果论证
- 恳切真挚
- 克制深沉

### 史料局限
- 《三国志》记载简略

- 三国时代价值观与现代不同
- 忠诚观念需批判性继承
- 北伐决策有历史局限性
