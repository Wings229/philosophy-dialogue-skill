# 诸葛亮研究信息来源

## 一手史料（最高优先级）

### 1. 《三国志》
- **作者**：陈寿（233-297）
- **成书时间**：西晋（约 280-290 年）
- **相关篇章**：
  - 《蜀书·诸葛亮传》
  - 《蜀书·先主传》
  - 《蜀书·后主传》
  - 《蜀书·马谡传》
- **可信度**：★★★★★
- **备注**：同时代史学家，最权威史料

### 2. 《诸葛亮集》
- **编者**：后人辑录
- **收录内容**：
  - 《出师表》
  - 《诫子书》
  - 《便宜十六策》（部分）
  - 《将苑》（部分）
  - 书信、奏表等
- **可信度**：★★★★☆
- **备注**：部分篇章真伪有争议

### 3. 《后汉书》
- **作者**：范晔（398-445）
- **成书时间**：南朝宋
- **可信度**：★★★★☆
- **备注**：补充东汉末年背景

## 二手史料（高优先级）

### 1. 《三国志注》
- **作者**：裴松之（372-451）
- **成书时间**：南朝宋
- **价值**：补充大量史料，引用已佚文献
- **可信度**：★★★★☆

### 2. 《资治通鉴》
- **作者**：司马光（1019-1086）
- **成书时间**：北宋
- **相关篇章**：汉纪、魏纪、蜀纪
- **可信度**：★★★★★
- **备注**：权威编年体史书

### 3. 《华阳国志》
- **作者**：常璩（约 291-361）
- **成书时间**：东晋
- **价值**：记载蜀地历史
- **可信度**：★★★★☆

## 研究原则

1. **一手优先**：优先采用《三国志》等一手史料
2. **多方印证**：重要事件需多个史料印证
3. **区分虚实**：严格区分史实与演义
4. **注明出处**：所有引用注明信息来源
5. **承认争议**：有争议处明确标注
