# 朱熹生平时间线（1130-1200）

## 早年（1130-1148）

| 年份 | 年龄 | 事件 |
|------|------|------|
| 1130 | 1 岁 | 出生于福建尤溪，父朱松，母祝氏 |
| 1135 | 6 岁 | 父朱松授以《孝经》 |
| 1138 | 9 岁 | 父朱松去世，临终托孤于胡宪、刘子翚、刘勉之 |
| 1139 | 10 岁 | 随母迁居福建崇安（今武夷山） |
| 1140-1147 | 11-18 岁 | 师从胡宪、刘子翚、刘勉之，学习儒家经典 |

**早年特点**：
- 幼年丧父，家道中落
- 三位老师都是理学家，奠定理学基础
- 勤奋好学，「自幼厉志圣贤之学」

---

## 求学与入仕（1148-1168）

| 年份 | 年龄 | 事件 |
|------|------|------|
| 1148 | 19 岁 | 考中进士（同榜有王十朋等） |
| 1149 | 20 岁 | 授左迪功郎、泉州同安主簿 |
| 1150 | 21 岁 | 赴同安任，兴学教化 |
| 1151-1153 | 22-24 岁 | 同安任满，返乡 |
| 1153 | 24 岁 | 赴延平拜李侗为师，正式学习理学 |
| 1154-1160 | 25-31 岁 | 多次拜见李侗，深受理学影响 |
| 1160 | 31 岁 | 李侗去世，朱熹成为理学传人 |
| 1162 | 33 岁 | 宋孝宗即位，上书言事 |
| 1163 | 34 岁 | 召对垂拱殿，向孝宗讲学 |
| 1164 | 35 岁 | 任武学博士，不久辞职 |
| 1166 | 37 岁 | 任南康军知军 |

**此期特点**：
- 进士及第，步入仕途
- 拜李侗为师，确立理学方向
- 仕途起伏，多次辞职

## 学术成熟期（1168-1181）

| 年份 | 年龄 | 事件 |
|------|------|------|
| 1168 | 39 岁 | 编撰《论语集注》《孟子集注》 |
| 1169 | 40 岁 | 完成《大学章句》《中庸章句》初稿 |
| 1170 | 41 岁 | 编撰《资治通鉴纲目》 |
| 1171 | 42 岁 | 任崇安社仓，推行社仓法 |
| 1172 | 43 岁 | 编撰《伊洛渊源录》 |
| 1173 | 44 岁 | 母亲去世，守丧 |
| 1174 | 45 岁 | 守丧期间，著述不辍 |
| 1175 | 46 岁 | **鹅湖之会**，与陆九渊辩论；与吕祖谦合编《近思录》 |
| 1176 | 47 岁 | 讲学于福建 |
| 1177 | 48 岁 | 完成《周易本义》 |
| 1178 | 49 岁 | 任知南康军 |
| 1179 | 50 岁 | **复兴白鹿洞书院**，制定学规 |
| 1180 | 51 岁 | 讲学白鹿洞书院 |
| 1181 | 52 岁 | 离南康军，提举浙东常平茶盐公事 |

**此期成就**：
- 完成《四书章句集注》主体
- 鹅湖之会，确立学术地位
- 复兴白鹿洞书院，影响深远
- 编撰多部重要著作

## 政治风波期（1182-1194）

| 年份 | 年龄 | 事件 |
|------|------|------|
| 1182 | 53 岁 | 弹劾台州知州唐仲友，得罪权贵 |
| 1183 | 54 岁 | 辞职，返乡讲学 |
| 1184 | 55 岁 | 与陈亮开始「王霸义利之辩」 |
| 1185 | 56 岁 | 讲学于福建 |
| 1186 | 57 岁 | 任江西提刑，不久辞职 |
| 1187 | 58 岁 | 任知漳州 |
| 1188 | 59 岁 | 在漳州推行经界法，受阻 |
| 1189 | 60 岁 | 完成《四书章句集注》定稿 |
| 1190 | 61 岁 | 任知潭州，修复岳麓书院 |
| 1191 | 62 岁 | 辞职，返乡 |
| 1192 | 63 岁 | 创建沧州精舍（考亭书院） |
| 1193 | 64 岁 | 任知漳州，再次推行经界法 |
| 1194 | 65 岁 | 任焕章阁待制兼侍讲，为宁宗讲学；不久被劾去职 |

**此期特点**：
- 仕途坎坷，多次被劾
- 坚持学术立场，不妥协
- 完成《四书章句集注》定稿
- 创建考亭书院

## 晚年与庆元党禁（1195-1200）

| 年份 | 年龄 | 事件 |
|------|------|------|
| 1195 | 66 岁 | 韩侂胄掌权，理学被斥为「伪学」 |
| 1196 | 67 岁 | 被革职，「伪学逆党」名单公布 |
| 1197 | 68 岁 | 党禁加剧，弟子受迫害 |
| 1198 | 69 岁 | 著《楚辞集注》，借屈原抒发情怀 |
| 1199 | 70 岁 | 病重，仍讲学不辍 |
| 1200 | 71 岁 | **三月九日病逝于建阳**，葬建阳唐石里 |

**临终遗言**：
- 「吾道不孤」
- 「辛苦一生，只为此事（理学）」

**身后**：
- 1202 年：党禁解除
- 1209 年：宋宁宗追谥「文」，称「朱文公」
- 1212 年：《四书章句集注》颁行学校
- 1241 年：从祀孔庙
- 1313 年：元仁宗恢复科举，以朱熹注为标准

## 生平数据

- **生卒**：1130-1200（71 岁）
- **籍贯**：徽州婺源（今江西婺源），生于福建尤溪
- **字号**：字元晦，号晦庵，晚年号晦翁
- **谥号**：文（朱文公）
- **官职**：最高任焕章阁待制兼侍讲（从四品）
- **著述**：《四书章句集注》《近思录》《朱子语类》等
- **弟子**：数千人，「朱门弟子」
- **影响**：理学集大成者，影响中国及东亚近千年
