# 他者视角：历代对黑格尔的评价

## 一、同时代人的评价

### 1. 谢林 (F.W.J. Schelling, 1775-1854)

**早期评价 **(1801-1803)：
- 视黑格尔为合作者
- 认为黑格尔追随自己的哲学

**后期批判 **(1841 年后)：
- 柏林讲演中公开批判黑格尔
- 指责黑格尔的"否定哲学"
- 提出"实证哲学"对抗黑格尔

**著名评价**：
- "黑格尔把一切存在都吞进概念的胃里"

### 2. 叔本华 (Arthur Schopenhauer, 1788-1860)

**激烈批判**：
- 称黑格尔为"笨拙的、令人作呕的、无知无识的江湖骗子"
- 指责黑格尔哲学是"伪哲学"
- 认为黑格尔的辩证法是"诡辩术"

**批判要点**：
- 黑格尔的语言晦涩是故意掩盖空洞
- 辩证法缺乏真正的逻辑基础
- 国家哲学是为普鲁士专制辩护

**背景**：
- 叔本华故意选择与黑格尔同一时间开课
- 结果学生都去了黑格尔那里
- 叔本华终生怀恨

### 3. 费尔巴哈 (Ludwig Feuerbach, 1804-1872)

**早期**：黑格尔追随者
**后期**：批判者

**批判要点**：
- 黑格尔哲学是"思辨神学"
- 绝对精神是上帝的哲学伪装
- 主张从抽象思维回到感性存在

**著名命题**：
- "上帝是人的本质的异化"
- "黑格尔哲学是神学的最后避难所"

## 二、19 世纪的评价

### 1. 马克思 (Karl Marx, 1818-1883)

**批判性继承**：

**肯定**：
- 辩证法的革命性
- 历史发展的观念
- 异化理论的洞见

**批判**：
- 黑格尔是"头足倒置"的
- 需要把辩证法"颠倒"过来
- 从唯心主义转向唯物主义

**著名评价**：
- "辩证法在黑格尔手中神秘化了，但这决没有妨碍他第一个全面地有意识地叙述了辩证法的一般运动形式"
- "我的辩证方法，从根本上来说，不仅和黑格尔的辩证方法不同，而且和它截然相反"

**继承的概念**：
- 异化劳动
- 历史辩证法
- 否定之否定

### 2. 恩格斯 (Friedrich Engels, 1820-1895)

**评价**：
- 黑格尔是"最伟大的哲学家"
- 黑格尔体系是"唯心主义的最高峰"
- 辩证法是黑格尔的"合理内核"

**贡献**：
- 系统阐述马克思主义对黑格尔的继承与批判
- 《路德维希·费尔巴哈和德国古典哲学的终结》

### 3. 克尔凯郭尔 (Søren Kierkegaard, 1813-1855)

**存在主义批判**：

**批判要点**：
- 黑格尔忽视了个体存在
- 体系哲学无法容纳生存的焦虑
- "真理是主体性"

**著名批判**：
- 黑格尔建造了宏伟的宫殿，自己却住在旁边的茅屋里
- 体系忘记了"存在着的个体"

**针对黑格尔的著作**：
- 《哲学片段》
- 《非科学的最后附言》

### 4. 尼采 (Friedrich Nietzsche, 1844-1900)

**批判**：
- 黑格尔是"德国庸人"的代表
- 辩证法是"奴隶道德"的哲学表达
- 历史哲学是"末人"的意识形态

**评价**：
- 肯定黑格尔的严肃性
- 但认为其哲学缺乏生命力

## 三、20 世纪的评价

### 1. 新黑格尔主义

**英国新黑格尔主义**：
- T.H. 格林 (T.H. Green)
- F.H. 布拉德雷 (F.H. Bradley)
- 伯纳德·鲍桑葵 (Bernard Bosanquet)

**特点**：
- 继承黑格尔的绝对唯心论
- 反对英国经验主义
- 后被分析哲学批判

### 2. 存在主义的评价

**海德格尔 **(Martin Heidegger)：
- 黑格尔是"西方形而上学的完成"
- 《精神现象学》是对存在历史的重要见证
- 但黑格尔仍未超越形而上学

**萨特 **(Jean-Paul Sartre)：
- 继承黑格尔的主奴辩证法
- 《存在与虚无》中的"他者"理论受黑格尔影响
- 但拒绝绝对精神的观念

### 3. 法兰克福学派

**阿多诺 **(Theodor Adorno)：
- 《否定的辩证法》
- 批判黑格尔的"同一性思维"
- 主张非同一性的辩证法

**霍克海默 **(Max Horkheimer)：
- 继承黑格尔的批判精神
- 但拒绝绝对真理的主张

**哈贝马斯 **(Jürgen Habermas)：
- 继承黑格尔的承认理论
- 发展交往行为理论
- 批判黑格尔的国家哲学

### 4. 法国黑格尔主义

**科耶夫 **(Alexandre Kojève)：
- 1930 年代巴黎黑格尔讲演
- 影响一代法国思想家（萨特、梅洛 - 庞蒂、拉康等）
- 强调主奴辩证法和历史终结

**著名解读**：
- 历史终结于普遍同质国家
- 黑格尔是马克思的先驱

### 5. 分析哲学的评价

**早期批判**：
- 罗素：黑格尔逻辑基于过时的亚里士多德逻辑
- 摩尔：批判黑格尔的内在关系说

**后期重新评价**：
- 皮平 (Robert Pippin)：黑格尔的现代性理论
- 布兰顿 (Robert Brandom)：黑格尔的推理主义
- 麦克道尔 (John McDowell)：黑格尔的自然主义

## 四、当代评价

### 1. 霍耐特 (Axel Honneth)

**承认理论**：
- 继承黑格尔的承认概念
- 发展为社会批判理论
- 《为承认而斗争》

### 2. 齐泽克 (Slavoj Žižek)

**拉康式黑格尔**：
- 黑格尔是拉康的先驱
- 辩证法与精神分析的结合
- 《少于无》

### 3. 平卡德 (Terry Pinkard)

**非形而上学解读**：
- 黑格尔不是传统形而上学家
- 黑格尔是社会规范性理论的先驱
- 《黑格尔的现象学：社会性理论》

## 五、评价的历史演变

| 时期 | 主导评价 | 代表人物 |
|------|----------|----------|
| 1830s-1840s | 分裂：青年/老年黑格尔派 | 施特劳斯、鲍威尔 |
| 1850s-1880s | 衰落：新康德主义兴起 | 李普曼、朗格 |
| 1890s-1920s | 复兴：英国新黑格尔主义 | 格林、布拉德雷 |
| 1930s-1950s | 批判：存在主义、分析哲学 | 海德格尔、罗素 |
| 1960s-1980s | 重新发现：法兰克福学派、法国理论 | 阿多诺、科耶夫 |
| 1990s-至今 | 多元解读：分析黑格尔主义、批判理论 | 平卡德、霍耐特 |
