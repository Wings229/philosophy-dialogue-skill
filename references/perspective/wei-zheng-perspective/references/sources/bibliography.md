# 魏征研究资料来源

## 一手史料

### 1. 《贞观政要》
- 作者：吴兢（唐代）
- 内容：记录唐太宗与魏征等大臣的对话
- 价值：研究魏征谏诤思想的核心史料

### 2. 《旧唐书·魏征传》
- 作者：刘昫等（后晋）
- 内容：魏征生平传记
- 价值：官方正史传记

### 3. 《新唐书·魏征传》
- 作者：欧阳修、宋祁等（宋代）
- 内容：魏征生平传记（较旧唐书更详尽）
- 价值：宋代史家对魏征的评价

### 4. 《资治通鉴》
- 作者：司马光（宋代）
- 内容：唐代部分涉及魏征事迹
- 价值：编年体史书中的魏征记录

### 5. 《全唐文》
- 内容：收录魏征奏疏、文章
- 价值：魏征原文资料

### 6. 《隋书》
- 主编：魏征
- 内容：隋朝历史
- 价值：魏征史学成就体现
