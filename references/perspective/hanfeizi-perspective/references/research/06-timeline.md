# 韩非子时间线

## 生平关键节点

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 约前 280 年 | 生于韩国贵族（韩公子） | 身处弱国，目睹韩国衰败，激发变法思想 |
| 约前 260 年 | 师从荀子（与李斯同门） | 吸收儒家礼法思想，但走向法家极端；荀子「性恶论」深刻影响韩非 |
| 约前 250-240 年 | 著《韩非子》55 篇 | 系统阐述法、术、势思想；因口吃不善言辞，以著述见长 |
| 约前 240 年 | 多次上书韩王不被采纳 | 写《孤愤》《说难》反映怀才不遇；「孤愤」之名由此而来 |
| 前 234 年 | 秦王嬴政读其书，叹「得见此人与之游，死不恨矣」 | 思想被秦始皇认可；为见韩非发动攻韩战争 |
| 前 233 年 | 出使秦国，被李斯陷害入狱，服毒自杀 | 法家内部斗争的牺牲品；李斯嫉妒其才，派人送毒药 |
| 前 221 年 | 秦始皇统一六国，全面推行法家制度 | 韩非思想在实践中达到顶峰；废分封、行郡县、书同文、车同轨 |
| 前 207 年 | 秦朝灭亡（距统一仅 15 年） | 法家思想的局限暴露：严刑峻法失人心，缺乏韧性 |

## 思想转折点

1. **师从荀子**：从儒家转向法家的关键
   - 荀子主张「性恶论」但认为可通过礼乐教化改善
   - 韩非极端化：人性自利不可改变，只能用制度约束

2. **上书不纳**：从理想主义者转向冷峻现实主义者
   - 多次向韩王进言不被采纳
   - 《孤愤》：「智法之士与当道之人，不可两存之仇」

3. **秦国之死**：法家思想的悖论
   - 韩非思想帮助秦国统一六国
   - 但韩非本人被法家同门（李斯）害死
   - 证明法家「术」的阴暗面：即使设计者也无法自保

## 后世影响时间线

| 时间 | 事件 |
|------|------|
| 前 206 年 - 公元 220 年（汉代） | 「独尊儒术，外儒内法」——对法家的修正 |
| 魏晋南北朝 | 法家思想式微，玄学兴起 |
| 唐代 | 李世民「水能载舟亦能覆舟」——儒家民本对法家的修正 |
| 宋代 | 王安石变法吸收法家思想（「天变不足畏，祖宗不足法」） |
| 明代 | 张居正改革：「循名责实」的法家方法 |
| 清代 | 雍正帝推崇法家，严刑峻法 |
| 近代 | 梁启超、章太炎重新评价法家 |
| 现代 | 冯友兰、萧公权、牟宗三等系统研究 |
