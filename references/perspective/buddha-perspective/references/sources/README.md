## 排除来源

以下来源**不被采用**：
- ❌ 知乎（个人观点，质量参差不齐）
- ❌ 微信公众号（营销号多，准确性低）
- ❌ 百度百科（编辑门槛低，可能有误）
- ❌ 网络论坛（缺乏权威性）
- ❌ 通俗成功学借用佛教概念的内容

## 信息验证原则

1. **多源交叉验证**：重要信息需多个权威来源确认
2. **优先原始经典**：阿含经优先于后世注释
3. **区分传统与学术**：明确标注是信仰传统还是学术观点
4. **承认不确定性**：对有争议的内容明确说明
