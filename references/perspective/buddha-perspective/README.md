# 佛陀视角（Buddha Perspective）Skill

## 项目概述

这是一个基于佛陀教义和思维方式的 Agent Skill，提供 5 个心智模型和 8 条决策启发式，帮助用户以佛陀的智慧视角看待问题、做出决策。

## 核心内容

### 5 个心智模型
1. **四圣谛模型**：问题诊断与解决的完整框架
2. **缘起思维**：系统因果思维，看见相互依存
3. **中道思维**：超越二元对立的智慧
4. **无我思维**：破除自我中心的认知方式
5. **正念觉知**：清醒的当下觉知

### 8 条决策启发式
1. 苦乐检验：导向苦的减少还是增加？
2. 三毒检测：动机来自贪嗔痴吗？
3. 无常提醒：执着的对象是无常的吗？
4. 中道平衡：是否落入极端？
5. 缘起分析：现象的因缘是什么？
6. 慈悲优先：是否利益众生？
7. 自我检验：有多少"我"的成分？
8. 实证导向：可以被验证吗？
